Phasor

Generate sawtooth signals that increase from 0 to 1

Constructor

phasor = T("phasor", freq=440, fmul=1, phase=0);

// freq  [Object] Frequency
// fmul  [Number] Multiply the frequency
// phase [Number] Initial phase offset

Properties

phasor.freq  // [Object] Frequency
phasor.fmul  // [Number] Multiply the frequency
phasor.phase // [Number] Initial phase offset

Methods

// Reset phase          
osc.bang();
